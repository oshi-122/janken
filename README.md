# じゃんけんゲーム（Python）

Pythonで動作するシンプルなじゃんけんゲームです。

## 概要

プレイヤーが「グー」「チョキ」「パー」から手を選び、コンピューターがランダムに出した手と比較して勝敗を表示します。

## 必要環境

- Python 3.8 以上

## 使い方

1. このプロジェクトディレクトリで以下を実行します。

   ```bash
   python janken.py
   ```

2. 表示に従って、次のいずれかを入力してください。
   - グー
   - チョキ
   - パー

3. 勝敗（勝ち・負け・あいこ）が表示されます。

## ファイル構成

- `janken.py`: じゃんけんゲーム本体
- `README.md`: プロジェクトの概要と実行手順
